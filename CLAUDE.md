# 概要
このリポジトリは、マークアップエンジニア／フロントエンドエンジニア向け ステップアップワークショップのための原稿です。

# 対象者
- マークアップ寄り業務が多いが、フロントエンドエンジニアの領域にも挑戦している方
- 自分で実装は行わないが、エンジニアと密にコミュニケーションを取って業務を進めているディレクターの方

# ワークショップ全体のゴール
- フロントエンド全般に関わるスキルセットの全体像を把握し、今の自分が取り組む領域を俯瞰できるようになり、その領域を更に拡大させる
